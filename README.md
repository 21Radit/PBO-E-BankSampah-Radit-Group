# Deskripsi Proyek

**Sistem Informasi Bank Sampah (SIBAS) - Fase 1**

Proyek ini bertujuan untuk mendigitalisasi pengelolaan bank sampah komunitas. Sistem ini dibuat untuk menggantikan pencatatan manual di buku besar sehingga proses pengelolaan data menjadi lebih rapi, cepat, dan akurat.

Dengan sistem ini, data nasabah, jenis sampah, serta transaksi setoran sampah dapat dicatat secara terstruktur sehingga meminimalkan kesalahan pencatatan dan mempermudah pengelolaan saldo nasabah.

# Spesifikasi Teknis (Sub-CPMK 1.1 - 1.4)

* **Nasabah**
  Mengelola data identitas nasabah serta menyimpan informasi saldo yang dimiliki oleh setiap nasabah.

* **Sampah**
  Mengelola jenis-jenis sampah yang dapat disetor oleh nasabah beserta harga per satuannya.

* **Transaksi**
  Mencatat transaksi setoran sampah dari nasabah yang menghubungkan data nasabah dengan jenis sampah tanpa terjadi redundansi data.
